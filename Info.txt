PL:
Duplicate Lines Finder
wersja 2.0

Autor: Mateusz "m4tx" M.
www.m4tx.pl

Wymagania: dowolny system obsługiwany przez Javę SE 1.6 lub nowszą; zainstalowane JRE w wersji 1.6 lub nowszej.

Program służy do wyszukiwania zduplikowanych linii w plikach tekstowych. Pozwala na wybranie separatora, który oddziela kolejne ciągi znaków.

EN:
Duplicate Lines Finder
version 2.0

Author: Mateusz "m4tx" M.
www.m4tx.pl/en

Requirements: any OS supported by Java SE 1.6 or later; JRE 1.6 or later installed.

An application is used to search duplicated lines in text files. It allows to choose delimiter, which separates subsequent character strings.
